Building on FastAPI and Svelte

I will try to base the blog on my attempts to build a platform for doing utilityprograms based on FastAPI as a backend and Svelte as a frontend. As most of the programs will run on an old fashioned Microsoft based ad I will have to let users authenticate through ldap with FastAPI. The database will be Postgresql and I will probably base talking to the database through the orm named ormar. Since the apps will be hosted on Kubernetes i will try to get a setup running docker both in development and in the test builds.
