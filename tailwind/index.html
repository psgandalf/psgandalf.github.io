Tailwindcss

I recently started to look at Tailwindcss. I usually make smaller utility programs and Tailwind seems like a good choice. Being able to use it with Svelte Kit would be really nice. Since Svelte Kit is new there where not that many blogs describing it. I found that svelte-add did have a tailwindscss addon. When I first added it I got this error. A couple of day later when I tested it again i did install. So keep in mind that this is beta stuff and may aswell fail in another couple of days...

First run

When I started the site created with npm init svelte@next sitename that is the way to install Svelte Kit and choosing to create a skeleton project with nothing else. Do a npm i to install and then run npx svelte-add tailwindcss --jit. Running npm run dev the webpage worked but without tailwind working. Comparing with blogs i added a asset directory under src and added yet another css directory with a tailwind.css file containing:

@import "tailwindcss/base";
@import "tailwindcss/components";
@import "tailwindcss/utilities";

In the routes folder I added a __layout.svelte file containing:

<script>
    import '/src/assets/css/tailwind.css'
</script>
  
<main>
   <slot />
</main>

And now tailwind was working atleast in dev and built for node.
